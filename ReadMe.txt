Nombre:
GoKorea

Objetivo:
Mostrar información relevante para la gente que le interesa
viajar a corea del sur, mostrando un poco los lugares turísticos
y restaurantes.

Descripcion de modelos:

Hotel:
Este modelo es para guardar la informacion de los hoteles en la BD
Tiene la informacion del nombre, las estrellas y la ubicacion

Reserva:
Con este modelo se guarda la reserva del hotel
Tiene los campos de nombre, apellido y fecha, tambien una seleccion
para elegir al hotel que quieres reservar

Restaurante:
En este modelo se guarda la información de los restaurantes
Los campos son nombre, ubicacion, y calificacion

Ciudad:
Se guarda la información de las ciudades
Usa los campos de nombre, habitantes, y superficie

Lugar:
Se guarda la información de los lugares
Usa los campos de nombre, ubicacion, entrada y el tipo de lugar que es

Avatar:
Este modelo se usa para guarda la foto de avatar que el usuario elige
